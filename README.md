# ionic-app

## 环境配置
### Nodejs
下载并安装Nodejs
官方的仓库比较慢，要找个快一点的仓库

### android studio
安装android studio
D:\app\android-studio\plugins\android\lib\templates
配置ANDROID_HOME环境变量

### 下载 android sdk
使用android studio，指定sdk存放位置，并下载android sdk的各种包

### 复制gradle模板
复制 D:\app\android-studio\plugins\android\lib\templates 这个文件夹到 D:\app\android\sdk\tools
